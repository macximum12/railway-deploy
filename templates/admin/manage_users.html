Security Information

  • Admin Exception: The 'admin' user is exempt from first-login password changes to prevent deployment lockouts. Ensure admin credentials are changed manually in production.
  • Session Timeout: User sessions automatically expire after 30 minutes of inactivity
  • First Login: New users (except admin) must change their password upon first login
  • Password Expiry: Passwords expire every 90 days and must be changed
  • Account Lockout: Accounts are locked after 5 failed login attempts

Access Levels

  • Administrator: Full access to everything - user management, system settings, all features
  • Content Manager: Add/edit/delete findings, access to activity logs, content management
  • Contributor: Add and edit findings, limited content creation permissions
  • Viewer: View-only access, cannot add/edit findings or access management features
